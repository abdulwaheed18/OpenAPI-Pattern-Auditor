About This Project

The OpenAPI Pattern Auditor is an open-source tool designed to address a common pain point for teams working with diverse technology stacks. An OpenAPI specification is a single source of truth, but its patterns are often implemented in multiple languages (Java, JavaScript, Go, Python, etc.), each with its own regex engine quirks.

This application helps by:

  • Validating Syntax: Checking if a regex pattern is syntactically correct for different language engines.
  • Identifying Quality Issues: Warning about patterns that are too permissive, lack proper anchors, or may be vulnerable to security risks like ReDoS.
  • Improving Collaboration: Providing a centralized place for developers, QAs, and security teams to audit and improve API contracts.
